Neural Network Description

ผมจะอธิบายกระบวนการ พัฒนาโมเดล Neural Network ตั้งแต่การเตรียมข้อมูล ไปจนถึงการเทรนและประเมินผล

1. การเตรียมข้อมูล (Data Preparation)

ก่อนเทรนโมเดลผมได้ใช้ข้อมูลภาพก้อนเมฆจาก Kaggle: Cloud Satellite Maxar Dataset เพื่อนำมาจำแนกเป็น Cloud (รูปภาพที่มีก้อนเมฆ) และ Not Cloud (รูปภาพที่ไม่ใช่ก้อนเมฆ).

หลังจากนั้นจะทำการแบ่งข้อมูลเป็น Train Set(80%) และ Validation Set(20%) ตามที่ผมได้เทรนไปครับ

2. ทฤษฎีของ Neural Network ที่พัฒนา

2.1 Convolutional Neural Network (CNN)

การแยกประเภทภาพ (Image Classification) ผมใช้ Convolutional Neural Network (CNN) ซึ่งประกอบด้วยเลเยอร์ Convolution, Pooling, และ Fully Connected เพื่อตรวจจับ Features ในภาพ เช่น ขอบ, รูปร่าง, และรายละเอียดต่าง ๆ. จากนั้นสรุปผล (Cloud / Not Cloud).

ขั้นตอนการทำงานของ CNN:

  • Convolution Layer: ใช้ Kernel เลื่อนผ่านภาพ
  • Pooling Layer: ลดขนาดภาพ
  • Flatten: แปลงข้อมูลเป็น 1 มิติ
  • Fully Connected Layer: ทำนายผลด้วย Neuron หลายตัว

3. ขั้นตอนการพัฒนาโมเดล Neural Network

  1. Data Loading: โหลดภาพจาก Dataset (Cloud / Not Cloud)
  2. Preprocessing: Resize, Normalize, Augmentation
  3. Model Architecture: เลือกใช้ CNN เป็นหลัก
  4. Compile Model: กำหนด Loss (เช่น binary_crossentropy), Optimizer (เช่น Adam)
  5. Train Model: กำหนด Epoch, Batch Size, แล้วฝึกโมเดล
  6. Evaluate: ประเมิน Accuracy, Loss บน Validation Set
  7. Save Model: บันทึกเป็นไฟล์ .keras (หรือ .h5) แต่ผมใช้.kerasเพราะในColabบอกว่าตัวนี้ดีกว่า.h5ครับ
