Machine Learning Description

Machine Learning & Neural Network เว็บไซต์นี้สาธิตการพัฒนาโมเดล Machine Learning และ Neural Network ที่ใช้สำหรับทำนายราคาบ้านใน California และการทำนายรูปภาพก้อนเมฆ โดยใช้ Colab ในการเทรนโมเดลครับ

1. การเตรียมข้อมูล (Data Preparation)

ก่อนการเทรนโมเดล เราได้รวบรวมข้อมูลจาก Kaggle: California Housing Prices. จากนั้นทำความสะอาดข้อมูล เช่น จัดการ Missing Values, ตรวจสอบ Outliers, และแปลงฟีเจอร์ให้พร้อมสำหรับการทำนาย

เมื่อข้อมูลพร้อม เราจึงแบ่งข้อมูลออกเป็น Train Set (ใช้ฝึกโมเดล) และ Test Set (ใช้ประเมินโมเดล) เพื่อให้แน่ใจว่าโมเดลสามารถทำนายได้ดีกับข้อมูลใหม่

2. ทฤษฎีอัลกอริทึม Machine Learning ที่ใช้

2.1 Linear Regression

Linear Regression เป็นอัลกอริทึมพื้นฐานสำหรับ Regression (การทำนายค่าตัวเลข) โดยมีสมมติฐานว่าความสัมพันธ์ระหว่าง Feature (X) กับ Target (Y) เป็นเส้นตรงในรูปสมการ Y = aX + b.

ขั้นตอนพัฒนา:

  • กำหนดค่าเริ่มต้นของน้ำหนัก (Weights) และ Bias
  • คำนวณค่าทำนายกับค่าจริง แล้วหาความผิดพลาด (Error)
  • ใช้วิธี Gradient Descent เพื่อลดความผิดพลาด
  • ปรับค่าพารามิเตอร์จนกว่าความผิดพลาดจะต่ำ

2.2 Gradient Boosting Regressor

Gradient Boosting เป็นเทคนิค Ensemble Learning ที่นำ Decision Tree หลาย ๆ ต้นมาต่อกันเป็นขั้น ๆ (Sequential) โดยแต่ละต้นจะพยายามแก้ไขข้อผิดพลาดของต้นก่อนหน้า

ขั้นตอนพัฒนา:

  • เริ่มจากโมเดลพื้นฐาน (เช่น ต้นไม้ตื้น ๆ)
  • คำนวณ Residual (ข้อผิดพลาด) ของโมเดลปัจจุบัน
  • ฝึกต้นไม้ถัดไปให้คาดการณ์ Residual เหล่านั้น
  • นำผลมารวมกัน (Boost) เพื่อปรับปรุงการทำนาย
  • ทำซ้ำจนกว่าจะครบจำนวนต้นไม้ หรือไม่มีการปรับปรุง

3. ขั้นตอนการพัฒนาโมเดลทั้ง 2 ประเภท (ML)

  1. Data Loading: โหลดข้อมูลจาก CSV (Kaggle)
  2. Data Cleaning: จัดการ Missing, Outliers และแปลงฟีเจอร์
  3. Train/Test Split: แบ่งข้อมูล 80%/20%
  4. Train Model:
    • Linear Regression: คำนวณค่าพารามิเตอร์ (Weight, Bias)
    • Gradient Boosting Regressor: ฝึกต้นไม้ทีละต้นเพื่อแก้ Residual ของต้นก่อนหน้า
  5. Model Evaluation: ประเมินด้วย Metrics เช่น MSE, RMSE, R² Score
  6. Save Model: บันทึกโมเดลเป็นไฟล์ .pkl เพื่อใช้งานต่อในvscode
